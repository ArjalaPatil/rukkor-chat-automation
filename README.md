# Rukkor Chat Automation

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- Eclipse
- Page Object Model (partial)

## Scenarios Covered
1. Login with valid credentials
2. Login with invalid credentials
3. Send & receive chat message (two users)
4. Edit message
5. Delete message (sender only)
6. Add reaction to message

## How to Run
1. Import project into Eclipse
2. Update Maven dependencies
3. Right-click testng.xml → Run As → TestNG Suite

## Assumptions / Limitations
- File upload automation not included due to unclear UI selector
- Stable test environment required
- Credentials are test accounts
